I'm a geodesy and cartography engineer living in Warsaw. Currently I work as a land surveyor but I'd like to change my line of work and take up new challenges as a front-end developer.

In June 2017 I decided to start learning web development. Since then I've completed a few online courses concerning not only the main technologies such as HTML, CSS and JavaScript but also jQuery, Bootstrap, Sass and LESS, Git, BEM methodology and web design. Creating my projects I put emphasis on making the code clear, semantic and responsive as well as the page look modern and highly esthetic. What I want to focus on now is to understand more deeply how JS works behind the scenes and how to use it in practice.

The important part of my life are travels and photography (combination of which you can see below). Travelling the world lets me improve interpersonal skills and broaden my horizons whereas the practical advantage of photographic hobby is ability to use graphic programmes like Photoshop. I'm also interested in astrology and space technology – it makes me breathless when Elon Musk's reusable rockets launch and than their boosters return after a few minutes.
